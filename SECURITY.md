# Security Policy

## Supported Versions

Only the latest release on the `main` branch receives security updates.

| Version | Supported |
|---------|-----------|
| latest (main) | ✅ |
| older | ❌ |

## Reporting a Vulnerability

**Please do not open a public GitHub issue for security problems.**

Report vulnerabilities privately by emailing **muhammadmirqobilov@gmail.com** with:

- A clear description of the issue
- Steps to reproduce (proof of concept if possible)
- Affected version/commit
- Your assessment of the impact

You will receive a response within **7 days** acknowledging the report. If the issue is confirmed, we will work on a fix and coordinate disclosure with you.

## Scope

In scope:
- Topso'z Android app source code in this repository
- Build and release tooling under `tools/`

Out of scope:
- Third-party dependencies (report upstream)
- Issues requiring a physically compromised device
- Publicly-known AdMob publisher IDs (these are designed to be visible in the shipped APK)

## Thanks

We appreciate responsible disclosure and will credit reporters in release notes unless anonymity is requested.
